Exemplos de Tradução para LIBRAS

Comparação entre texto original e texto adaptado

Exemplo 1 — Algoritmo

Texto original

Um algoritmo é uma sequência finita de passos bem definidos que resolvem um problema computacional.

Texto adaptado para LIBRAS

Algoritmo é uma sequência de passos.

Os passos seguem uma ordem.

O objetivo é resolver um problema.

O problema é computacional.

Exemplo 2 — Variável

Texto original

Uma variável é um espaço na memória do computador utilizado para armazenar valores que podem mudar durante a execução do programa.

Texto adaptado para LIBRAS

Variável é um espaço na memória.

Esse espaço guarda um valor.

O valor pode mudar.

A mudança acontece durante o programa.

Exemplo 3 — Estrutura condicional

Texto original

Estruturas condicionais permitem que um programa execute diferentes ações dependendo de uma condição ser verdadeira ou falsa.

Texto adaptado para LIBRAS

Estrutura condicional é uma decisão.

O programa testa uma condição.

A condição pode ser verdadeira.

A condição pode ser falsa.

O programa executa ações diferentes.
